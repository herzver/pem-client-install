## 这是什么

明镜 PEM 不是任务看板，也不是替代财务记账的软件。

它解决的是广告公司项目交付过程中的经营核算问题：

- 这个客户、这个项目、这个部门到底赚没赚钱。
- 外协供应商、物料、拍摄、搭建、后期等成本有没有及时归集。
- 开票、回款、付款是否都挂到了对应项目。
- 年框客户、Campaign、活动执行和制作项目能不能分开核算利润。
- 老板、财务、AE 和执行是否能看同一本项目经营账。

## 客户端和网页版有什么区别

功能和数据是同一套。

- Windows 客户端：适合习惯桌面软件操作的用户。

客户端需要联网使用，不支持离线。这样可以保证多端同步，也避免项目数据散落在单台电脑里。
